Dental Clinic Appointment Management System

  • PHP
  • Laravel
  • Supabase
  • Web-Based
  • SMS Integration
  • Responsive Design


I independently designed, developed, and deployed a full-stack dental clinic appointment management system. The application features a public landing page that showcases the clinic’s information, an appointee dashboard for booking appointments, and dedicated dashboards for administrators and dentists to manage and track appointments in real time. The system allows users to schedule appointments, monitor their booking status, and view details about the attending dentist.
