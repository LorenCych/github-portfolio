CLAmBook: Campus Library Attendance Management SYstem

CLAMBook is a comprehensive library attendance management system consisting of two WinForms applications. One application handles administrative tasks such as attendance monitoring, management, and report generation, while the other serves as a self-service kiosk for logging attendance using RFID technology. These applications communicate over a shared network, ensuring a smooth and efficient attendance tracking system.

Technologies Used

  • C#
  • Windows Forms
  • .NET 8
  • Microsoft SQL Server Express
  • Networking

Features

Admin and Kiosk Application

The Admin Application features a dashboard for real-time attendance monitoring, tracking top library visitors, analyzing the top 10 most active college programs, and visualizing visit trends through interactive charts.

The Kiosk Application enables fast and convenient self-check-ins using RFID scanning.

Fast and Convenient Attendance Logging via RFID Cards

Admins can assign RFID cards to MinSU students for seamless check-ins.

The Kiosk Application allows quick tap-and-go logging for a frictionless experience.

Formattable, Printable, and Exportable Reports

Generate, format, print, and export attendance reports in daily, weekly, or monthly views.

Equipped with Pivot Table functionality for cleaner numerical representation and insightful data analysis.

Secure App Access

Password Recovery System to eliminate access issues and prevent workflow disruptions.

Automatic User-Logging System to track who accesses the Admin Application for added security.
