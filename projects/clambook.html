CLAmBook: Campus Library Attendance Management SYstem

CLAMBook is a comprehensive library attendance management system consisting of two WinForms applications. One application handles administrative tasks such as attendance monitoring, management, and report generation, while the other serves as a self-service kiosk for logging attendance using RFID technology. These applications communicate over a shared network, ensuring a smooth and efficient attendance tracking system.

HOW CLAMBOOK MAKES A DIFFERENCE?

Admin & Kiosk Applications

The Admin Application features a dashboard for real-time attendance monitoring, tracking top library researchers, analyzing the top 10 most active college programs, and visualizing visit trends through graphical charts.

The Kiosk Application enables fast and convenient self-check-ins using RFID scanning. Additionally, it is equipped with on-screen keyboards and inputs to still facilitate library time-ins for non-RFID holders and other visitors.

Fast & Convenient Attendance Logging via RFID Cards

Library admins can assign RFID cards to MinSUans for seamless time-ins.

The Kiosk Application allows quick tap-and-go logging for a frictionless experience. It is also equipped with error-preventive functions to handle invalid inputs or errors.

Formattable, Printable, & Exportable Reports

The system is able to generate, format, print, and export attendance reports in daily, weekly, or monthly views.

Additionally, it is equipped with Pivot Table functionality for cleaner numerical representation and insightful data analysis.

Essential Data-Management Controls

Apart from the typical CRUD (create, read, update, delete) capabilities. Library admins can take advantage of bulk-add and bulk-delete actions for seamless data migration and clean-up.

Such functions are vital for managing data of RFID Card holders.

Secure App Access & User Monitoring

Equipped with Password Recovery System to eliminate access issues and prevent workflow disruptions.

Automatic User-Logging System to track who accesses the Admin Application for added layer of security.
