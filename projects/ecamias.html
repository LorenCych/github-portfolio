ECamias Companion Application

ECamias is an Android application that works alongside our Automated Camias Dryer and Slicer, letting users control the physical machine, track its current process and run-time, monitor its internal drying chamber environment, and stay updated through real-time notifications.

How ECamias works together with our Physical Machine?

The physical Automated Camias Dryer and Slicer and the ECamias app stay connected through Firebase. Commands sent from the app, like starting a slicing batch or pausing a drying cycle, reach the machine in real time, while sensor data from the drying chamber, including temperature, humidity, weight, and process status, is pushed back to the app to keep both systems in sync.

Real-Time Process Tracking

The Home dashboard displays a horizontal progress stepper that highlights exactly which phase the machine is currently in, slicing, drying, or weighing, so users always know where the process stands at a glance. A short status message and dynamically updated control buttons, such as Start Slicing or Pause and Proceed to Weighing, let the app reflect the machine's actual state.

Live Parameter Monitoring

The Live Monitoring panel streams the drying chamber's temperature, humidity, current weight, and elapsed run-time straight from the machine's sensors. These readings update continuously while a batch is processing, giving users a real-time view of conditions inside the chamber without needing to check the machine in person.

Remote Machine Control

Users can start a new slicing batch, pause an ongoing drying cycle, or proceed to the weighing phase, all directly from the app. Because these controls only appear when relevant to the machine's current phase, the interface stays simple while still giving users full control over the process from a distance.

Categorized Notification Feed

Every phase, batch starts, phase transitions, completions, or errors, is logged as its own entry in the Notifications tab, tagged as Success, Error, or Info and filterable by date through a built-in calendar picker. This gives users a second way to stay on top of the machine's status beyond the home screen, especially useful for checking on a batch that finished while they were away.

Historical Reading
Graphs

The Previous Readings page turns past sessions into interactive line graphs, letting users view temperature and humidity independently or overlaid on the same chart for easier comparison. A detailed log history beneath the graph breaks each reading down by timestamp, weight, temperature, and humidity, and both the graph and log can be printed directly from the app.

Secure Application
Controls

Critical actions, like starting a new batch or stopping an ongoing process, trigger a confirmation pop-up before executing, preventing accidental taps from disrupting a running cycle. A dedicated pause button lets users temporarily halt the machine mid-process, while a separate emergency button is available to immediately stop operations if something goes wrong, giving users a fast, reliable way to intervene when needed.
